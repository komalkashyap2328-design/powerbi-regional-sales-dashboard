# powerbi-regional-sales-dashboard
Interactive Power BI dashboard analyzing regional revenue, profit, and margin performance.

## Dashboard Preview
![Dashboard Screenshot](Screenshot%202026-02-25%20133413.png)

## Project Overview
This interactive Power BI dashboard analyzes regional sales performance across revenue, profit, and profit margin.

## Tools Used
- Power BI
- Data Modeling
- DAX Measures
- Data Visualization
- Business Insight Generation

## Key Insights
- West region leads in revenue, profit, and margin.
- Central region shows lowest margin (7.9%), indicating cost/pricing inefficiencies.
- Margin variation highlights optimization opportunities.
